Grant Thornton R&D Onboarding Tool

Welcome to the Grant Thornton R&D Onboarding Tool! This guide is intended to help new team members get familiar with our processes.

Throughout this guide, four icons will remain consistent. These being:

  • The question mark icon: Lost your way around the system? Click the question mark icon on any page for help on how to navigate the system.
  • The home icon: Click the home icon on any page to return to this homepage where you can pick a new journey.
  • The pencil icon: Looking to update the contents of a page? Provided you have administrator privileges, you can click the pencil and begin making changes.
  • The speaker icon: The default contents of each page can be narrated by clicking the speaker icon.

To get started, click the ‘Next’ link below, or to jump to a specific section, select from the boxes on the left.
